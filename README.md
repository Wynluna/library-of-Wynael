# ᛞ̇ Ma Bibliothèque — Guide d'installation

## Déployer sur Vercel (gratuit, 5 min)

### 1. GitHub
1. Va sur github.com → "New repository" → nomme-le `ma-bibliotheque`
2. Clique "uploading an existing file"
3. Glisse-dépose **tout le contenu** de ce dossier
4. "Commit changes"

### 2. Vercel
1. Va sur vercel.com → "Add New Project"
2. Sélectionne ton repo `ma-bibliotheque`
3. Laisse tout par défaut → **Deploy**
4. ~1 minute → tu reçois un lien ex: `ma-bibliotheque.vercel.app` 🎉

### 3. Écran d'accueil
**iPhone (Safari) :**
→ Partager → "Sur l'écran d'accueil" → Ajouter

**Android (Chrome) :**
→ Menu ⋮ → "Ajouter à l'écran d'accueil" → Ajouter

---
✦ Une fois hébergée, la recherche de livres par titre/auteur fonctionnera !
